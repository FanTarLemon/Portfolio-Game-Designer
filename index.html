Bubblow

Role: Game Designer & UI

The Challenge (48 Hours)

ออกแบบและสร้างเกม 2D Shooter ที่ต้องแข่งกับเวลา (Global Game Jam) โจทย์คือสร้างเกมเพลย์ที่เรียนรู้ง่ายแต่มีความท้าทายซ่อนอยู่

Execution
  • Mechanics Design: ออกแบบกติกาหลัก พาฟองสบู่ขึ้นสู่ผิวน้ำ พร้อมระบบบริหารพื้นที่และขนาดตัวที่ได้รับผลกระทบจากการโจมตีศัตรู
  • Game Balancing: ปรับจูนความยากของศัตรูแต่ละเฟสอย่างรวดเร็ว เพื่อให้เกมสนุกตลอดระยะเวลาการเล่น
  • Team Communication: ประสานงานและแจกแจงงานกับทีม Art และ Dev ภายใต้ความกดดัน เพื่อให้รวมโค้ดทันส่งใน 48 ชม.

Turn-Based Strategy AI Behavior UI Wireframe

Furry War

Role: Game Designer

Design Goal

ออกแบบโครงสร้างเกมเพลย์ Turn-Based Strategy ที่ผู้เล่นต้องคำนวณทิศทาง แรงยิงวิถีโค้ง (Projectile) และใช้สภาพแวดล้อมให้เป็นประโยชน์ พร้อมระบบ AI ที่ตัดสินใจตามสถานการณ์

My Contribution
  • Core Mechanics & UI: ร่าง Wireframe หน้าจอ UI ต่างๆ (Main Menu, Character Select) และกำหนด Flow การเล่นหลัก
  • AI Behavior System: ออกแบบ Logic พฤติกรรมบอท (Decision Making) แยกเป็นโหมด Normal, Combat และ Escape เพื่อตอบโต้ผู้เล่นอย่างเป็นระบบ
  • Combat System: วางเงื่อนไขระบบการเล็งเป้าหมาย การยิงแบบมีโอกาสพลาด (Random) และระบบสลับเทิร์น (Winning Streak)
